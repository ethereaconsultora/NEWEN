# 🌿 Newen

> *"Lo que sentís, puede ser acompañado. Buscanos."*

Plataforma que conecta personas en momentos difíciles con counselors humanistas verificados.

---

## ¿Qué es Newen?

Newen (mapuche: "fuerza, espíritu") es una PWA que funciona como:
- **Red profesional** para counselors verificados por la AAC.
- **Plataforma de ayuda** para personas que buscan acompañamiento psicológico y emocional.
- **Beneficio corporativo** para empresas que cuidan la salud mental de sus equipos.

---

## Stack

| Capa | Tecnología |
|---|---|
| Frontend | Next.js 15 + React 19 + TypeScript 5 + Tailwind CSS 4 |
| Backend | Next.js API Routes + Supabase |
| Base de datos | Supabase PostgreSQL + Auth + RLS |
| Videollamada | Daily.co |
| Pagos | Mercado Pago |
| Email | Resend |
| Deploy | Vercel |
| PWA | Sin App Store · Sin Google Play |

---

## Roles

| Rol | Acceso |
|---|---|
| **Consultante** | Buscar counselors, reservar sesiones ($22 USD), videollamada, evaluar |
| **Counselor** | Perfil profesional, agenda, métricas, comunidad, talleres |
| **Admin** | Gestión total: postulaciones, entrevistas, empresas, sponsors |

---

## Desarrollo

```bash
# Instalar dependencias
npm install

# Desarrollo local
npm run dev

# Build
npm run build
```
